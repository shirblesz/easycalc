# 🎙️ TalkCalc

**Calculator for Everyone** — simple, accessible, and beautiful.

## What is TalkCalc?

TalkCalc is a calculator designed to be truly easy to use. Big buttons, voice input, and a clear display — whether you're 8 or 80, TalkCalc just works.

## ✨ Features

- 🎤 **Voice Input** — say "twenty five plus thirty" and get your answer
- ⌨️ **Text Input** — type naturally like "234 * 39" or "234 times 39"
- 👆 **Extra-Large Buttons** — easy to tap, hard to miss
- 🔊 **Voice Output** — hear every action and result spoken aloud
- 📳 **Haptic Feedback** — feel a vibration on every press
- 🎨 **5 Themes** — Midnight Blue, Rosewood, Forest, Lavender, High Contrast
- 🔤 **3 Text Sizes** — Normal, Large, Extra Large
- 🔗 **Chain Calculations** — type "- 7" after a result to keep going
- 📴 **Works Offline** — no internet needed after install
- 🔒 **Private** — no ads, no tracking, no data collection

## 🎤 Voice & Text Commands

Speak or type naturally:

| You say / type | TalkCalc does |
|---|---|
| "25 plus 30" | 25 + 30 = 55 |
| "twelve times four" | 12 × 4 = 48 |
| "50 divided by 5" | 50 ÷ 5 = 10 |
| "234 * 39" | 234 × 39 = 9126 |
| "- 7" (after result) | continues from previous result |
| "clear" | resets calculator |

## 🛠️ Built With

- Vanilla JavaScript (zero dependencies)
- Progressive Web App (PWA)
- Atkinson Hyperlegible Font (designed for low vision)
- Web Speech API
